# Restricted Text Area

### This project displays the basic use of input event listeners for Input Elements

## Features:
* Uses HTML 5, CSS, and JavaScript
* Implements **Event Listeners and Input Max Lengths**
* Type into the text area and you can see the text at the bottom right updating live
* A function is called when the input reaches the max length of chars 

## How to Run:

This project can run locally within your computer without installation.

* Clone or Download this Repository.
    ```bash 
    git clone https://github.com/jFiesta11/Roadmap.sh-Frontend-Project-6---Restricted-Textarea.git
    ```

* Double Click the `index.html` file to open in your default browser or right click the `index.html` file and click on 'open' or 'open with' and select the browser to open this file with.
